      <h2 class="topic-title">📅 Weekly Setup & Time Management</h2>
      <p class="topic-intro">
        The week can fly by before you realize you’ve forgotten a quiz or missed a reading.
        A strong weekly setup helps you stay on top of deadlines and frees up mental space
        to focus on learning — not just surviving…or passing a course.
      </p>
      <p class="topic-intro">
        Starting your week with intention sets the tone for everything else. Weekly planning isn’t just about fitting things into your calendar — it’s about deciding what matters most and aligning your tasks with your academic and personal goals.
      </p>

      <div class="content-block text-gray-300">
        <h2 class="section-title">💡 Why Weekly Planning Works</h2>
        <ul class="list-disc list-inside mb-4 ml-4 space-y-2">
          <li><strong>🧠 Free Up Mental Bandwidth</strong><br>
            When you plan ahead, your brain doesn’t have to juggle every task at once. It can focus on learning — not remembering deadlines.
          </li>
          <li><strong>🔁 Spot the Patterns</strong><br>
            Seeing your whole week laid out helps you notice problems early. Are your Mondays too packed? Do you always forget a quiz on Thursdays? Your calendar tells the story.
          </li>
          <li><strong>🚨 Avoid Last-Minute Panic</strong><br>
            Weekly planning cuts down on “Wait — that’s due today?!” moments. Instead of scrambling, you’ll have space to stay on track.
          </li>
        </ul>

        <h3 class="text-xl font-semibold mb-3">🗓️ How to Plan Your Week (Step-by-Step)</h3>

        <h4 class="text-lg font-semibold mb-2">1. Pick a Weekly Planning Day</h4>
        <p class="mb-2"><strong>🧠 Why it matters:</strong> Your week needs a starting line. Without one, everything blends together.</p>
        <ul class="list-disc list-inside mb-4 ml-4 space-y-2">
          <li>Choose Sunday — it’s a natural reset day.</li>
          <li>Set a timer for 15–20 minutes.</li>
          <li>Make it a ritual: use coffee, candles, or a playlist to help you focus.</li>
        </ul>

        <h4 class="text-lg font-semibold mb-2">2. Review Your Course Syllabus and Set Up Your Calendar</h4>
        <p class="mb-2"><strong>🧠 Why it matters:</strong> Most school stress is predictable — your course syllabus already has all the major dates.</p>
        <ul class="list-disc list-inside mb-4 ml-4 space-y-2">
          <li>Go through each course and add big dates (exams, assignments, quizzes) into your calendar.</li>
          <li>Color-code each course. Example:
            <ul class="list-disc list-inside mt-2 ml-4">
              <li>History Lecture = Dark Blue</li>
              <li>History Tutorial = Light Blue</li>
            </ul>
          </li>
          <li>Add recurring events for every class and tutorial.</li>
          <li>Ask yourself:
            <ul class="list-disc list-inside mt-2 ml-4">
              <li>What’s due this week?</li>
              <li>Are there any rolling tasks (like labs or discussion posts)?</li>
              <li>What needs to be started early?</li>
            </ul>
          </li>
        </ul>
        <p class="mb-4"><strong>🧠 Pro Tip:</strong> Keep one master calendar with both school and life stuff. It saves brainpower.</p>

        <h4 class="text-lg font-semibold mb-2">3. Set Weekly Priorities</h4>
        <p class="mb-2"><strong>🧠 Why it matters:</strong> Some weeks are light. Others are chaos. You need to know what’s coming so you can plan around it.</p>
        <ul class="list-disc list-inside mb-4 ml-4 space-y-2">
          <li>Look at all the deadlines and events for the week.</li>
          <li>Create a daily to-do list based on what’s coming.</li>
          <li>Ask: What 2–3 things must get done this week?</li>
          <ul class="list-disc list-inside mt-2 ml-4">
            <li>Big assignment?</li>
            <li>Group project?</li>
            <li>Job interview?</li>
            <li>Mental health check-in?</li>
          </ul>
        </ul>
        <p class="mb-4"><strong style="color:#ff4d4f;">⚠️ If two or more big things are due this week, don’t add extra shifts at work or plan a Netflix binge. Prioritize smart.</strong></p>
        <p class="mb-4"><strong>🎯 Planning = trading short-term “meh” for long-term ease. You don’t need to cancel fun — just schedule it smarter.</strong></p>

        <h4 class="text-lg font-semibold mb-2">4. Time Block Like a Pro</h4>
        <p class="mb-2"><strong>🧠 Why it matters:</strong> If it’s not in your calendar, it doesn’t exist. Time blocking removes the guesswork from your day.</p>
        <ul class="list-disc list-inside mb-4 ml-4 space-y-2">
          <li><strong>Start with non-negotiables:</strong> class times, tutorials, labs, assignment deadlines</li>
          <li><strong>Add essentials:</strong> sleep, meals, commute, gym, work</li>
          <li><strong>Schedule study time:</b> same time each week if possible</li>
          <li><strong>Block chores:</strong> groceries, cleaning</li>
          <li><strong>Include downtime:</strong> it’s not optional — it’s maintenance</li>
        </ul>
        <p class="mb-2"><strong>🎨 Suggested color coding:</strong></p>
        <ul class="list-disc list-inside mb-4 ml-4 space-y-2">
          <li>Courses: Different color for each</li>
          <li>Tutorials/Labs: Lighter version of the course color</li>
          <li>Study: Yellow</li>
          <li>Gym: Green</li>
          <li><li>Social: Purple</li>
          <li>Chores: Orange or Brown</li>
          <li>Meals/Sleep: Grey or light blue</li>
          <li>Downtime: Teal or something calming</li>
        </ul>
        <p class="mb-4"><strong>🧩 Think of your calendar like a puzzle. Start with what’s fixed, then fill in the rest to balance.</strong></p>

        <h4 class="text-lg font-semibold mb-2">5. Rotate Your Resources</h4>
        <p class="mb-2"><strong>🧠 Why it matters:</strong> You’re human — you can’t max out every area of life at the same time.</p>
        <p class="mb-2">Think of your week like a pie chart:</p>
        <ul class="list-disc list-inside mb-4 ml-4 space-y-2">
          <li>Study 🧠</li>
          <li>Sleep 😴</li>
          <li>Gym 💪</li>
          <li>Work 💼</li>
          <li>Social 👯</li>
        </ul>
        <p class="mb-2">✅ Ask:</p>
        <ul class="list-disc list-inside mb-4 ml-4 space-y-2">
          <li>Which areas matter most this week?</li>
          <li>Which ones can shrink temporarily?</li>
        </ul>
        <p class="mb-4"><strong>📌 Example:</strong> If you have two major essays due, your gym and social time may shrink — and that’s okay. You’ll bring them back next week.</p>

        <h4 class="text-lg font-semibold mb-2">6. Plan Backwards — Not Just for This Week</h4>
        <p class="mb-2"><strong>🧠 Why it matters:</strong> School isn’t just about what’s due now. Planning ahead gives your future self a break.</p>
        <ul class="list-disc list-inside mb-4 ml-4 space-y-2">
          <li>The moment a paper or test is announced, start planning prep blocks that same week.</li>
          <li>Break the big task into steps:
            <ul class="list-disc list-inside mt-2 ml-4">
              <li>Research</li>
              <li>Outline</li>
              <li><li>Write</li>
              <li>Edit</li>
            </ul>
          </li>
          <li>Add each step to your calendar — not just your to-do list.</li>
        </ul>
        <p class="mb-4"><strong>🧠 This turns scary deadlines into small, doable chunks.</strong></p>
        <p class="mb-4"><strong>📌 Pro Tip:</strong> Knowing what’s on your plate each week shows you where you have free time — and where you don’t. That clarity helps you say “yes” or “no” to plans without guilt.</p>

          <li><strong>🗂️ Google Calendar/Outlook Calendar</strong><br>Color-code your classes, study time, chores, gym, and social time. Use alerts and recurring events to build habits.</li>
          <li><strong>📘 Notion</strong><br>Great for students who love visuals. Use it for organizing syllabus, study plans, and to-do lists in one spot.</li>
          <li><strong>📒 OneNote / Todoist</strong><br>Good combo for note-taking and task management. Extra useful if your school offers Microsoft 365.</li>
          <li><strong>📝 Google Docs / Word</strong><br>Works just fine. Create folders and headers so your files don’t disappear into digital chaos.</li>
          <li><strong>🎙️ Lecture Recordings</strong><br>Recording (with permission) can help with review. Just note: auto-transcription tools may miss details and glitch — useful, but not perfect.</li>
        </ul>
        <p class="mb-4"><strong>💡 Bottom line:</strong> Whether it’s paper or digital, build your system early — before Week 3 chaos hits.</p>

        <h3 class="text-xl font-semibold mb-3">🚫 Common Mistakes to Avoid</h3>
        <ul class="list-disc list-inside mb-4 ml-4 space-y-2">

          <li>⏳ <strong>No Buffer Time</strong><br>Add 15–30 minute breaks between big tasks. Use them for snacks, walks, or catching up.</li>
          <li>📧 <strong>Ignoring School Email</strong><br>Many students miss important updates just because they don’t check their inbox.</li>
        </ul>
        <p class="mb-4"><strong>✅ Fix it fast:</strong><br>
          Most schools offer Microsoft 365 for free — check if yours does.<br>
          Download the Outlook app on your phone and laptop.<br>
          Turn on notifications so you never miss a message.</p>

      <h2>🧠 Planning Daily with Intention</h2>
      <p><strong>Mastering realistic time-blocking and daily scheduling</strong></p>

      <h3>💡 Why This Matters</h3>
      <p>Even with weekly plans, your day can fall apart if you don’t zoom in. Planning daily helps you stay flexible, prioritize based on energy, and actually finish what matters — without burning out.</p>

      <h3>🧠 Why Daily Planning Feels Hard (But Matters More Than You Think)</h3>
      <ul>
        <li>They don’t know where to start</li>
        <li>They think they don’t have time</li>
        <li>They’ve failed at it before</li>
      </ul>
      <p>But the real issue? Without a plan, your time gets eaten by everyone else’s priorities. This is how you take it back.</p>

      <h3>🛠 Step-by-Step: How to Time-Block Your Day Like a Pro</h3>

      <h4>🕘 1. Start with a Blank Daily View</h4>
      <p>Use your digital calendar or a printable daily planner to map out your full day. Block out what’s fixed first — here are some examples:</p>
      <ul>
        <li>✅ Classes <em>(If you completed the Weekly Setup & Time Management section, your class schedule is already in your calendar)</em></li>
        <li>✅ Commute</li>
        <li>✅ Job shifts</li>
        <li>✅ Appointments</li>
      </ul>
      <p><strong>🧠 Why this matters:</strong> These are your non-negotiables — the parts of your day that don’t move. Once those are in place, you can see what time is truly available.</p>
      <p><strong>💡 Reminder:</strong> In the Weekly Setup & Time Management section, we focused on building daily to-do lists centered on coursework. But this step is about your full day — including personal and life tasks like laundry, appointments, or errands that still require time and attention. Don’t leave those out.</p>
      <p><strong>📌 Note:</strong> This isn’t an exhaustive list — just examples to help you start.</p>

      <h4>🍽️ 2. Add Real-Life Stuff Next (Be Honest)</h4>
      <table>
        <thead><tr><th>Task</th><th>Average Time Block</th></tr></thead>
        <tbody>
          <tr><td>Meals</td><td>20–30 min</td></tr>
          <tr><td>Shower + prep</td><td>30 min</td></tr>
          <tr><td>Gym</td><td>1–1.5 hours</td></tr>
          <tr><td>Chores</td><td>30–60 min</td></tr>
          <tr><td>Breaks</td><td>15–20 min chunks</td></tr>
          <tr><td>Social scroll</td><td>15–30 min</td></tr>
        </tbody>
      </table>
      <p><strong>🧠 Why this matters:</strong> Underestimating everyday tasks is one of the top reasons students fall behind. Build your day around reality, not fantasy.</p>

      <h4>📚 3. Estimate Study Time Based on Task Type</h4>
      <table>
        <thead><tr><th>Task Type</th><th>Suggested Time Block</th></tr></thead>
        <tbody>
          <tr><td>Readings</td><td>30–60 min</td></tr>
          <tr><td>Review notes</td><td>20–30 min</td></tr>
          <tr><td>Quizzes</td><td>30–45 min</td></tr>
          <tr><td>Research paper</td><td>2–3 hours (split up)</td></tr>
          <tr><td>Group work</td><td>1–1.5 hours</td></tr>
          <tr><td>Practice problems</td><td>45–60 min</td></tr>
        </tbody>
      </table>
      <p><strong>📌 Note:</strong> These blocks will change as your semester progresses and you build a rhythm. Don’t be afraid to adjust, especially if you’ve underestimated.</p>

      <h4>🎨 4. Stick to Color Coding</h4>
      <p>We cover color-coding in the Weekly Setup & Time Management section — this is your reminder to stay consistent. The more you stick to your system, the faster your brain reads your schedule. It’s a habit that pays off in speed and clarity.</p>

      <h4>🧩 5. Plan in “Energy Zones,” Not Just Time Slots</h4>
      <ul>
        <li><strong>Morning Boost (9am–12pm):</strong> High-focus work</li>
        <li><strong>Afternoon Slump (1–4pm):</b> Admin or low-focus work, maybe a gym break</li>
        <li><strong>Evening Wind Down (7–10pm):</strong> Review, reset, light prep</li>
      </ul>
      <p><strong>⚡ Pro Tip:</strong> Match tasks to your natural energy, not just your availability.</p>

      <h4>✅ 6. Review and Update Your Plan (Takes 2 Minutes)</h4>
      <ul>
        <li>✅ Look at your daily to-do list (built during your weekly planning session)</li>
        <li>✅ What got done?</li>
        <li>✅ ✅ What needs to move — to tomorrow or later in the week?</li>
      </ul>
      <p><strong>🧠 This connects your daily plan back to your weekly priorities</strong>, so nothing gets lost and your brain stays clear.</p>

    document.addEventListener("DOMContentLoaded", function() {
      const collapsibles = document.querySelectorAll(".collapsible");
      collapsibles.forEach(btn => {
        btn.addEventListener("click", function () {
          // Toggle the active class on the button
          this.classList.toggle("active");

          // Get the next sibling element, which is the content div
          const content = this.nextElementSibling;

          // Get the icon element within the button
          const icon = this.querySelector(".collapsible-icon");

          // Toggle the display of the content
          if (content.style.display === "block") {
            content.style.display = "none";
            if (icon) icon.textContent = "+"; // Change icon to plus when closed
          } else {
            content.style.display = "block";
            if (icon) icon.textContent = "-"; // Change icon to minus when open
          }
        });
      });

      // Show/hide Back to Top button based on scroll position
      const backToTopButton = document.getElementById('backToTopBtn');
      window.onscroll = function() {
        if (document.body.scrollTop > 20 || document.documentElement.scrollTop > 20) {
          backToTopButton.style.display = "block";
        } else {
          backToTopButton.style.display = "none";
        }
      };
    });

    function showTopic(topicId) {
        // Hide all topic sections
        const topicSections = document.querySelectorAll('#topic-content .topic-page');
        topicSections.forEach(section => {
            section.style.display = 'none';
        });

        // Show the requested topic section
        const selectedTopic = document.getElementById(topicId);
        if (selectedTopic) {
            selectedTopic.style.display = 'block';
        }

        // Hide the main content and show the topic content container
        document.getElementById('main-content').style.display = 'none';
        document.getElementById('topic-content').style.display = 'block';
        window.scrollTo(0, 0); // Scroll to top
    }

    function goBackToModule(moduleTarget) {
      // Hide all topic sections in case one is open
      const topicSections = document.querySelectorAll('#topic-content .topic-page');
      topicSections.forEach(section => {
          section.style.display = 'none';
      });

      // Hide the specific topic content container
      document.getElementById('topic-content').style.display = 'none';
      // Show the main content
      document.getElementById('main-content').style.display = 'block';
      // Scroll to the top of the page or specific module
      if (moduleTarget === 'main-content') {
        window.scrollTo(0, 0);
